wxflore
**wxflore** Electronic flora written in Python based on wxPython library

## Usage
wxflore-x.x/wxflore.py

$HOME/.wxflore/config.py
Exemple:
```
flore_root = "/home/git/wxflore/Flores/Main"
flore_img_path = "/Bota/Flores/Main/img"
```
